# cargo-audit snap

This repository contains the snap manifest for [cargo-audit](https://github.com/rustsec/rustsec/tree/main/cargo-audit),
a tool by the Rust Secure Code Working Group to audit crates and binaries for known security issues.

Please note that in order to properly this tool, you must use `cargo-audit audit ...options...`.
Running `cargo-audit ...options...` will give an error message.
I am fairly sure that this is not fixable, but I'd love to be proven wrong!
